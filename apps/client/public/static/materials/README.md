Place authorable texture maps here for use in editor `mapUrl` material slots.

Use URLs like:
- `/static/materials/cab/metal-panel.png`
- `/static/materials/corridor-door/brushed-frame.jpg`
- `/static/materials/stairwell/concrete.webp`
- `/static/materials/shared/checker.png`
- `/static/materials/shared/board-formed-concrete.svg`

The editor scans this folder tree and shows discovered textures in the material dropdowns.
Supported sample extensions: `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`.
